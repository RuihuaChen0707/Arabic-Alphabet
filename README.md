# 📚 阿拉伯语学习系统

一个现代化的阿拉伯语学习平台，采用循序渐进的学习路径设计，帮助学习者系统掌握阿拉伯语基础。

## 🌟 项目特色

- 🎯 **完整学习路径**：从字母表到高级课程的6阶段学习体系
- 🎴 **智能闪卡系统**：AI生成插片的单词卡片，带发音和转写
- 💰 **成本优化**：图片缓存机制，避免重复API调用
- 📱 **移动端友好**：完美适配手机和平板设备
- 🔄 **离线支持**：SVG备选图片，无网络也能学习

## 🚀 功能概览

### 📖 第一课 - 字母表
- 28个阿拉伯语字母完整展示
- **字母发音**：点击播放标准阿拉伯语发音
- **字母形态**：独立、开头、中间、结尾形式
- **发音符号**：开口符、齐齿符、合口符及长音变体
- **手写练习**：支持触摸屏书写练习
- **测试模式**：多种题型检验学习效果

### 🎴 第二课 - 认读单词闪卡
- **20个基础词汇**：日常生活常用阿拉伯语单词
- **AI生成图片**：每个单词配备独特AI插图
- **标准发音**：内置阿拉伯语语音播放
- **罗马音转写**：帮助学习者掌握发音规则
- **例句展示**：每个单词包含实用例句
- **学习模式**：
  - 学习模式：顺序学习所有单词
  - 复习模式：重点练习未掌握单词
- **进度追踪**：标记困难单词和已掌握单词

- 第四课 - 日常对话
- 第五课 - 阅读理解

## 🛠️ 技术架构

### 前端技术
- **HTML5**：语义化结构，支持离线存储
- **CSS3**：响应式设计，流畅动画效果
- **JavaScript**：模块化代码，现代ES6+语法
- **Web Speech API**：阿拉伯语语音合成
- **CSS Grid/Flexbox**：响应式布局

### AI集成
- **OpenRouter API**：统一的AI服务接口
- **Google Gemini 2.5 Flash Image**：高质量图片生成
- **智能提示词**：专门为教育场景优化的提示词
- **图片缓存系统**：避免重复调用，节省成本

### 缓存优化
- **图片缓存**：每个单词的图片只生成一次
- **本地存储**：学习进度和个人设置保存
- **离线图片**：SVG备选图片确保功能完整

## 📱 使用指南

### 开始学习
1. 打开应用，进入主页学习路径
2. 选择要学习的课程（推荐按顺序学习）
3. 跟随指导完成学习内容

### 第一课使用方法
1. **浏览字母**：点击任意字母查看详细信息
2. **学习发音**：点击🔊按钮听取标准发音
3. **练习书写**：在白板区域练习字母书写
4. **测试模式**：进入测试检验学习效果

### 第二课使用方法
1. **浏览单词**：使用左右按钮切换单词
2. **查看详情**：点击"翻转卡片"查看中文意思和用法
3. **学习发音**：点击🔊按钮听取单词发音
4. **标记进度**：
   - 点击"困难"标记需要多练习的单词
   - 点击"已掌握"标记熟练的单词
5. **重新生成**：对不满意图片可点击🔄重新生成

### 学习建议
- 📅 **每日练习**：建议每天学习15-30分钟
- 🔄 **定期复习**：使用复习模式巩固已学内容
- 📊 **关注进度**：查看主页学习统计数据
- 🎯 **循序渐进**：完成当前课程再进入下一阶段

## 🔧 开发环境

### 本地运行
```bash
# 克隆项目
git clone https://github.com/RuihuaChen0707/Arabic-Alphabet.git

# 进入项目目录
cd Arabic-Alphabet

# 启动本地服务器
python3 -m http.server 8000

# 在浏览器中访问
open http://localhost:8000
```

### 项目结构
```
Arabic-Alphabet/
├── index.html          # 主页面结构
├── style.css           # 样式文件
├── script.js           # 核心逻辑
├── safari-polyfill.js   # Safari兼容性补丁
├── vercel.json         # Vercel部署配置
└── README.md           # 项目文档
```

## 💡 核心功能详解

### 图片生成系统
1. **AI生成**：Gemini 2.5 Flash Image创建教育插图
2. **缓存机制**：自动缓存生成的图片避免重复调用
3. **降级策略**：
   - AI生成 → Unsplash图片 → SVG图标 → 文字显示
4. **重新生成**：用户可选择重新生成不满意图片

### 语音系统
1. **阿拉伯语合成**：Web Speech API实现
2. **移动端优化**：Safari兼容性特殊处理
3. **备用方案**：文字视觉反馈作为语音失败备选

### 学习统计
- **已完成课程**：自动追踪课程完成情况
- **学习单词数**：记录学习过的单词总数
- **学习时间估算**：基于学习进度的学习时长估算
